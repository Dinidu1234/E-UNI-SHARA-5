# KickBlastUltraLight

WPF .NET 8 single-project application for athlete management and monthly fee calculations.

## Default Login
- Username: `dinidu`
- Password: `123456`

## Features
- Ribbon icon navigation (no sidebar text layout)
- Top search + context bar
- Right slide-in drawer for forms/details
- Floating action button behavior per page
- SQLite with ADO.NET (`Microsoft.Data.Sqlite`)

## Run
1. Open `KickBlastUltraLight.sln` in Visual Studio 2022.
2. Build and run the `KickBlastUltraLight` startup project.
